# SmartShop Management System

A shop management system developed using Python, Flask, SQLite and Bootstrap.

## Features

- Product Management
- Customer Management
- Sales Tracking
- Analytics Dashboard
- Settings Management
- Responsive UI

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap

## Installation

1. Clone repository

```bash
git clone https://github.com/codewithlb/SmartShop-Management-System.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run application

```bash
python app.py
```
